# 🌴 Fantasy Map Portfolio

---

# 🌍 English Version

## 📖 Overview
This project is a **personal portfolio website** with a unique and immersive concept:  
The homepage is an **interactive fantasy map**, where each island represents a different section of the site.  
Instead of a traditional navigation menu, the user explores the map and clicks on the islands to access the content.

---

## 🗺 Concept

### Homepage — Interactive Map
- **Fantasy map viewed from above** with a tropical ocean gradient background.
- **Four main islands**, each representing a portfolio section:
  1. **About Me** — Wooden cabin, campfire, books, cozy atmosphere.
  2. **Projects** — Gears, tools, creative constructions.
  3. **Blog** — Wizard tower, medieval library, scattered scrolls.
  4. **Contact** — Lighthouse, harbor, anchored ship.
- **Additional elements**:
  - Small boats sailing between islands.
  - Gentle wave animations.
  - Playful, colorful vector-inspired style with soft shadows.

---

## 📂 Site Structure

### Pages
1. **Home** — Interactive map with clickable islands.
2. **About Me** — Introduction text, photo, career timeline, skills.
3. **Projects** — Project list with images, descriptions, and links.
4. **Blog** — Post list, each displayed as a partially unrolled scroll.
5. **Post Page** — Parchment-textured background, serif typography, formatted content.
6. **Contact** — Functional form and social media links.

---

## 📜 Blog Design
- **Theme**: Medieval scribe’s desk or fantasy library.
- **Post cards**: Scrolls with slightly worn edges, warm beige tones.
- **Post view**: “Unrolling” animation when opening a scroll.
- **Typography**: Decorative medieval-style titles, readable serif body text.

---

## 🎨 Visual Style
- **Art style**: Playful, colorful, vector-illustration inspired.
- **Color palette**: Vibrant yet harmonious, with good contrast.
- **Animations**:
  - Hover effects on islands (slight zoom, tooltip with section name).
  - Smooth page transitions.
  - Optional ambient sounds (sea, wind, clicks) with toggle button.

---

## 🛠 Technical Notes
- The map can be implemented using **SVG**, **Canvas**, or **WebGL/Three.js** — technology choice is still open.
- The site must be **responsive** and performance-optimized.
- Modular and reusable code structure.
- SEO-friendly and compatible with modern browsers.

---

## 🚀 Goals
- Create an **interactive and memorable portfolio** that stands out from traditional layouts.
- Provide a **fun navigation experience** without losing usability and accessibility.
- Showcase creativity, technical skills, and attention to detail.

---

## 📌 Next Steps
1. Define visual assets (map, islands, scrolls).
2. Choose rendering approach (SVG, Canvas, or WebGL).
3. Build the interactive homepage with navigation.
4. Develop internal pages while maintaining theme and style.
5. Optimize for performance and publish.

---

## 💡 Inspirations

- Illustrated children’s books.
- Pirate treasure maps.
- Medieval manuscripts and scrolls.

# 🇧🇷 Versão em Português

## 📖 Visão Geral
Este projeto é um **site de portfólio pessoal** com um conceito único e imersivo:  
A página inicial é um **mapa de fantasia interativo**, onde cada ilha representa uma seção diferente do site.  
Em vez de um menu de navegação tradicional, o usuário explora o mapa e clica nas ilhas para acessar o conteúdo.

---

## 🗺 Conceito

### Página Inicial — Mapa Interativo
- **Mapa de fantasia visto de cima** com fundo de oceano tropical em degradê de azuis.
- **Quatro ilhas principais**, cada uma representando uma seção do portfólio:
  1. **Sobre Mim** — Cabana de madeira, fogueira, livros, atmosfera aconchegante.
  2. **Projetos** — Engrenagens, ferramentas, construções criativas.
  3. **Blog** — Torre de mago, biblioteca medieval, pergaminhos espalhados.
  4. **Contato** — Farol, porto, barco ancorado.
- **Elementos adicionais**:
  - Pequenos barcos navegando entre as ilhas.
  - Animações suaves de ondas.
  - Estilo lúdico, colorido, inspirado em vetores, com sombras suaves.

---

## 📂 Estrutura do Site

### Páginas
1. **Home** — Mapa interativo com ilhas clicáveis.
2. **Sobre Mim** — Texto de apresentação, foto, linha do tempo da carreira, habilidades.
3. **Projetos** — Lista de projetos com imagens, descrições e links.
4. **Blog** — Lista de posts, cada um exibido como um pergaminho parcialmente desenrolado.
5. **Página de Post** — Fundo com textura de pergaminho, tipografia serifada, imagens e conteúdo formatado.
6. **Contato** — Formulário funcional e links para redes sociais.

---

## 📜 Design do Blog
- **Tema**: Mesa de escriba medieval ou biblioteca de fantasia.
- **Cards de post**: Pergaminhos com bordas levemente desgastadas, tons quentes de bege.
- **Visualização do post**: Animação de “desenrolar” o pergaminho ao abrir.
- **Tipografia**: Títulos decorativos em estilo medieval, corpo do texto com fonte serifada legível.

---

## 🎨 Estilo Visual
- **Estilo artístico**: Lúdico, colorido, inspirado em ilustrações vetoriais.
- **Paleta de cores**: Vibrante, mas harmoniosa, com bom contraste.
- **Animações**:
  - Efeitos de hover nas ilhas (leve zoom, tooltip com nome da seção).
  - Transições suaves entre páginas.
  - Sons ambientes opcionais (mar, vento, cliques) com botão para ativar/desativar.

---

## 🛠 Notas Técnicas
- O mapa pode ser implementado usando **SVG**, **Canvas** ou **WebGL/Three.js** — a escolha da tecnologia está em aberto.
- O site deve ser **responsivo** e otimizado para performance.
- Estrutura de código modular e reutilizável.
- SEO-friendly e compatível com navegadores modernos.

---

- Proporcionar uma **experiência de navegação divertida** sem perder a usabilidade e acessibilidade.
